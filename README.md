# SkinGarden

## ¿Qué es SkinGarden?

Software de tienda online especializado en productos de cuidado de la piel, que permite a los usuarios realizar pedidos de manera rápida y eficiente. La plataforma incluye funcionalidades de gestión de productos, como agregar, editar y eliminar artículos, disponibles para usuarios con rol de empleado, garantizando un manejo organizado del catálogo.

## Problematica

Este proyecto se realizo en base de una problematica que decia: Una empresa de productos de cuidado corporal enfrenta una serie de inconvenientes por la falta de un software intuitivo que obstaculiza la eficiencia y experiencia del usuario. Entre estos problemas se encuentra la redundancia de datos,  proceso de pago en línea ineficiente, modificación de productos y además de una ausencia de un método de compra rápida. 

## Solucion de la problematica

En base a la problematica dada, pensamos en una solucion que dice: Desarrollar un software personalizado para la tienda en línea que implemente un sistema de pago de forma segura y fácil que admita una variedad de métodos, introducir opciones de carrito de compras con la funcionalidad de guardar información para futuras compras, incluso pueda realizar compras de manera segura y pueda modificar productos añadidos. Además de crear una plataforma que permita la fácil publicación y gestión de productos con validaciones para evitar repetir datos.
